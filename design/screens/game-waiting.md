## Elements

| Element | Description | Behavior |
|---------|-------------|----------|
| BackBtn | "вҶҗ Quay lбәЎi" | Leave room, return to lobby |
| RoomCode | 6-digit display + copy btn | Click copies to clipboard |
| PlayerSlot | Avatar circle with name | Filled = joined, empty = waiting |
| StartBtn | "BбәҜt Д‘бә§u" | Host only вҖ” starts game when 2+ players |
| ShareLink | Link display + copy button | Auto-copies share URL |
| WaitingMsg | "Дҗang chб»қ ngЖ°б»қi chЖЎi..." | Shows when < 2 players |

## States

- **Default**: Waiting for players. Start button disabled until 2+ joined.
- **Ready**: 2+ players joined. Start button enabled (host only).
- **Starting**: Host clicked Start вҖ” button shows spinner + "Дҗang bбәҜt Д‘бә§u..."
- **Full**: All 4 slots filled вҖ” "PhГІng Д‘ГЈ Д‘бә§y" shown.
- **Error**: Error toast on failure to start.

## Key Interactions

- **Host clicks "BбәҜt Д‘бә§u"** вҶ’ POST /api/games вҶ’ `game_started` Socket event вҶ’ all clients transition to Playing state
- **Copy link** вҶ’ `navigator.clipboard.writeText` вҶ’ toast "ДҗГЈ copy!"
- **Player joins** вҶ’ `room_update` event вҶ’ player slots update in real-time
- **Click on empty slot** вҶ’ no action (can't drag into slot)
